Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

  1. Web 2.0: Se refiere a la segunda generación de servicios de la web que permite la interacción y colaboración entre usuarios, facilitando la creación y el intercambio de contenido. Ejemplos incluyen redes sociales, blogs y wikis.
  2. Web 3.0: También conocida como la web semántica, busca mejorar la interoperabilidad y la comprensión de la información a través de la inteligencia artificial, el uso de datos estructurados y la personalización de experiencias.
  3. La W3C y los estándares: La World Wide Web Consortium (W3C) es una comunidad internacional que desarrolla estándares para asegurar que la web sea accesible, interoperable y evolutiva. Sus estándares incluyen HTML, CSS, y muchas otras tecnologías web.

II. Lenguajes de etiquetas

  1. SGML: (Standard Generalized Markup Language) es un lenguaje de marcado que define una estructura para documentos. Es la base de muchos lenguajes de marcado modernos.
  2. XML: (Extensible Markup Language) es un lenguaje de marcado que permite definir tus propias etiquetas. Es ampliamente utilizado para el intercambio de datos entre aplicaciones.
  3. HTML: (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido en la web.
  4. XHTML: (Extensible HyperText Markup Language) es una versión más rigurosa de HTML que cumple con las reglas de XML. Se diseñó para ser más flexible y compatible con otros lenguajes de marcado.
  5. HTML5: Es la versión más reciente de HTML, que incluye nuevas características para multimedia, gráficos y formularios, mejorando la semántica y la accesibilidad.

III. Otros lenguajes relacionados al desarrollo web

  1. CSS/CSS3: (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. CSS3 incluye nuevas características como transiciones y animaciones.
  2. JavaScript: Es un lenguaje de programación que permite implementar funcionalidades complejas en las páginas web. Permite crear interactividad y manipular el DOM.
  3. Ajax: (Asynchronous JavaScript and XML) es una técnica que permite la actualización de páginas web de forma asíncrona, lo que significa que se puede cargar información sin tener que recargar toda la página.
  4. Tecnologías y lenguajes del lado del servidor: Incluyen lenguajes como PHP, Python y Ruby, que permiten la generación de contenido dinámico en el servidor antes de enviarlo al navegador del cliente.
  5. Navegadores: Son aplicaciones que permiten a los usuarios acceder y visualizar páginas web. Ejemplos incluyen Google Chrome, Mozilla Firefox y Safari.
  6. Motores renderizados: Son componentes de los navegadores que interpretan el código HTML, CSS y JavaScript para mostrar la página web al usuario.

IV. Herramientas de desarrollo

  1. Software WYSIWYG: (What You See Is What You Get) son editores que permiten a los desarrolladores crear contenido web sin necesidad de escribir código, mostrando una representación visual de la página.
  2. Otras alternativas:
    1. Figma: Herramienta de diseño colaborativo para crear interfaces de usuario.
    2. InVisión: Plataforma de diseño de prototipos que permite a los equipos trabajar juntos en el diseño de experiencias digitales.
    3. Zeplin: Herramienta que facilita la colaboración entre diseñadores y desarrolladores, permitiendo exportar diseños a código.
    4. GitHub: Plataforma de desarrollo colaborativo que permite el versionado de código y la gestión de proyectos.
  3. Inspección de código desde el navegador: Herramienta de desarrollo que permite a los usuarios examinar el código fuente de una página web, facilitando la depuración y el análisis de rendimiento.
